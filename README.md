# Meus Filmes e Jogos Favoritos 2026

Projeto desenvolvido em **HTML e CSS puro**, com o objetivo de criar uma página simples e responsiva reunindo alguns dos meus filmes e jogos favoritos.

A proposta do projeto é praticar conceitos básicos de desenvolvimento Front-End, como estruturação de conteúdo, uso de imagens, criação de cards, organização visual e responsividade.

## 🔗 Página publicada

Acesse o projeto publicado pelo GitHub Pages:

[Ver página online](https://b4sileus-py.github.io/filmes-jogos-favoritos-2026/)

## 📌 Sobre o projeto

Esta página funciona como um pequeno catálogo pessoal, separando os conteúdos em duas categorias principais:

* Filmes favoritos
* Jogos favoritos

Cada item é apresentado em formato de card, contendo imagem, título, gênero/plataforma e uma breve descrição pessoal.

O projeto também utiliza um efeito visual nos cards de jogos, onde a imagem principal pode ser substituída por um GIF ao passar o mouse, simulando uma prévia da jogabilidade.

## 🎯 Objetivo

O principal objetivo deste projeto foi praticar:

* Estruturação de páginas com HTML
* Uso de tags semânticas
* Organização de conteúdo em seções
* Criação de cards com imagens
* Estilização com CSS externo
* Responsividade para diferentes tamanhos de tela
* Efeitos visuais com `:hover`
* Uso de imagens locais e GIFs externos

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* GitHub Pages

Durante o desenvolvimento deste projeto, pratiquei a criação de uma página visualmente organizada usando apenas HTML e CSS.

* Responsividade com media queries
